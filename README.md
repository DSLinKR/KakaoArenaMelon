## 재현 방법
1. main에 있는 python 파일들과 train, test 데이터셋을 같은 경로에 저장합니다.
2. 해당 경로에서 다음 코드를 terminal에 시행합니다.

<pre>
<code>
python inference.py
</code>
</pre>

3. 파일들이 포함된 경로에 결과파일이 *results.json* 으로 저장되어있을 것입니다.
